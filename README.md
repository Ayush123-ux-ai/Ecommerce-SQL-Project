# Ecommerce-SQL-Project
## Project Overview

This project is a complete E-Commerce SQL Analysis designed to understand how an online shopping platform operates using real-world style data. The database includes customers, products, orders, payments, and reviews, allowing you to explore the full sales cycleâ€”from purchase to customer feedback.

## Database Details

The project starts with building a fully normalized relational database consisting of six core tables:

Customers: User profiles with demographic and signup information.

Products: Catalog of all items with prices, categories, and stock levels.

Orders: High-level order information with Order Dates and order status.

Order Items: Line-level transaction details including product quantity and pricing.

Payments: Records of order payments, methods used, and payment amounts.

Reviews: Customer feedback, ratings, and review Date.

## Key Business Insights

Electronics category drives strong revenue based on price and demand.

High-value customers contribute significantly to total revenue.

Late reviews help identify customer dissatisfaction signals.

Category-wise sales performance shows which product lines perform best.

Returned orders with payments help detect refund mismatches.
